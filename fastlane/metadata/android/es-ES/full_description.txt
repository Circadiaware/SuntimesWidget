<p>Muestra los calendarios solares y lunares. Incluye una aplicación y una colección de widgets.</p>
<p>Muestra el amanecer, atardecer, crepúsculo (civil / náutico / astronómico), horas azul y dorada, solsticios y equinoccios, salida y puesta de la Luna, fases lunares e iluminación de la Luna.</p>
<p>Los tiempos mostrados por la aplicación son aproximados. La aplicación no requiere conexión a una red, el GPS es opcional.</p>
Algunos permisos son necesarios para que algunas opciones de la aplicación funcionen correctamente:
<ul>
<li><small><u>ACCESS_FINE_LOCATION</u></small> se utiliza para obtener la posición actual (latitud, longitud, altitud).</li>
<li><small><u>BOOT_COMPLETED</u></small>.</li>
<li><small><u>SET_ALARM</u></small> se utiliza para configurar alarmas utilizando la aplicación Reloj.</li>
<li><small><u>VIBRATE</u></small>.</li>
<li><small><u>WRITE_EXTERNAL_STORAGE</u></small> se utiliza para hacer copias de seguridad a un archivo (lugares, temas, etc.) <small>(sólo api<=18)</small>.</li>
</ul>
